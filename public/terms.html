Terms of Service of Parallel


This page lists all of the rules that must be followed while using Parallel. Violations of these rules may result in a user or a server being banned from using Parallel. Bans may be appealed or limited to a duration of time. By using Parallel, you agree to these terms of service. If you do not, please refrain from using the bot. If you have any questions regarding the terms of service, please contact us through the support server.



The following are NOT allowed:

  • Misconduct: Using Parallel as an aid to carrying out activities that are either illegal or against the Discord Terms of Service. A server with Parallel that performs misconduct (Parallel not directly being used to perform misconduct) may also be subject to being banned from Parallel.
  • Abuse: Abusing Parallel in any way, the act of intentionally trying to cause problems to the bot. This includes API spam.
  • Defamation: Intentionally defaming Parallel based off false claims.
  • Impersonation: Disguising another bot as Parallel to trick users into believing that it is the official Parallel bot.
  • License Infringement: Using the Parallel source code and violating the license.
