Parallel

An advanced moderation bot loaded with plenty of modern-moderation features capable of delivering precise moderation.

Invite

Features

Appeals

You are able to create an appeal for all types of infractions with Parallel, even bans (if they are enabled in the server.) If you are not in the server, you are able to create an appeal via Parallel's DM's.

Logging

Parallel supports both message logging and moderation logging.

Locking

Easily lock any designated channels quickly with just one command.

Tickets

Create private tickets for ticket support members.

Additional Infraction Information

Want to add extra information to all infractions but don't want to include them in the reason everytime? Use this! A message is customizable for each type of infraction.

Shortcuts

Shortcuts are shortcuts to moderation commands only. As the name suggests, they are a shortcut to a punishment command with a predefined reason and duration. They are also slash command compatible.

And so much more.
